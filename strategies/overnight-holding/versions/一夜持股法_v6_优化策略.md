# 一夜持股法 v6.0 - 优化版选股策略

## 一、核心优化点

### 1. 排除周五规则（新增）
- **规则**：周四收盘后（14:45-15:00）不执行选股
- **原因**：一夜持股法周五买入需承担周末2天不确定性风险
- **执行**：判断当前星期几，如果是周四，直接返回"今日周四，跳过选股，避免周末持仓风险"

### 2. 大盘环境过滤（强化）
- 上证指数 > MA20 且 上证涨幅 > 0%（原策略已有）
- **新增**：创业板指 > MA20
- **新增**：沪深300 > MA20
- **新增**：市场宽度 > 0.9（原策略已有）

### 3. 板块效应过滤（新增）
- 实时热点板块：涨幅 > 3% 的板块
- 个股必须属于热点板块成分股
- 优先选择板块内涨停数 ≥ 3 的板块

### 4. 评分系统（新增）
综合评分 = 资金得分 × 0.3 + 技术得分 × 0.3 + 情绪得分 × 0.2 + 板块得分 × 0.2

#### 资金得分（0-100）
- 主力资金净流入：> 1亿 = 100分，> 5000万 = 80分，> 1000万 = 60分，> 0 = 40分
- 大单买入占比：> 50% = 100分，> 30% = 80分，> 20% = 60分
- 集合竞价成交额：> 1亿 = 100分，> 5000万 = 80分，> 2000万 = 60分
- 集合竞价量比：> 5 = 100分，> 3 = 80分，> 2 = 60分

#### 技术得分（0-100）
- MACD：0轴上方且红柱放大 = 100分，0轴上方 = 80分，0轴附近 = 60分
- KDJ：已金叉 = 100分，即将金叉（J值>K值且差距<5）= 80分
- 均线：5>10>20且多头排列 = 100分，站上20日线 = 80分
- 布林带：中轨上方且接近上轨 = 100分，中轨上方 = 80分

#### 情绪得分（0-100）
- 集合竞价涨幅：2%-3% = 100分，3%-4% = 90分，1%-2% = 70分
- 昨日涨幅：3%-5% = 100分，5%-6% = 90分，2%-3% = 70分
- 昨日换手率：5%-8% = 100分，8%-12% = 90分，3%-5% = 70分
- 20天涨停次数：1-2次 = 100分，3-5次 = 80分，0次 = 60分

#### 板块得分（0-100）
- 所属板块涨幅 > 5% = 100分
- 所属板块涨幅 3%-5% = 80分
- 所属板块涨幅 < 3% = 50分
- 板块内涨停数 ≥ 5 = +20分
- 板块内涨停数 ≥ 3 = +10分

### 5. 排除条件优化
- 排除近3天有大股东减持公告（原策略）
- **优化**：排除近1天有负面新闻 → 改为"排除公告日期为当天的减持、业绩暴雷等利空公告"
- 排除昨日换手率 > 15%（原策略）
- 排除集合竞价涨停后又打开的股票（原策略）
- **新增**：排除ST、*ST、退市股（原策略已有）
- **新增**：排除上市不足60天的新股
- **新增**：排除当天有大宗交易折价 > 5% 的股票

## 二、执行流程（6步）

### 步骤1：判断交易日（新增）
- 判断今天是星期几
- 如果是周四（星期4），直接返回提示并结束
- 如果是周五、周六、周日，不执行（A股休市或即将休市）

### 步骤2：大盘环境检查（强化）
- 获取上证指数、创业板指、沪深300的实时行情
- 检查是否均在MA20之上
- 检查市场宽度是否 > 0.9
- **任一条件不满足，终止选股**

### 步骤3：板块扫描（新增）
- 获取所有板块实时涨幅排名
- 筛选出涨幅 > 3% 的热点板块
- 记录板块内涨停数量

### 步骤4：初筛股票池
- 非ST、非停牌、无退市风险
- 市值 30亿 - 150亿
- 股价 5元 - 30元
- 上市天数 > 60天

### 步骤5：技术指标筛选
- 资金指标：集合竞价成交额 > 5000万，量比 > 3，主力资金净流入 > 0，大单买入占比 > 30%
- 情绪指标：集合竞价涨幅 2%-4%，昨日涨幅 3%-6%，昨日换手率 5%-12%，20天内有过1-2次涨停
- 趋势指标：股价站上5日、10日、20日均线，MACD在0轴上方且红柱放大，KDJ金叉或即将金叉，布林带中轨上方运行

### 步骤6：综合评分与排序
- 对通过初筛的股票计算综合评分
- 按综合评分降序排序
- 取Top 10
- 生成推荐理由（基于得分最高的维度）

## 四、自动化配置

### 定时任务设置
- **执行时间**：每个交易日 14:30（收盘前30分钟）
- **判断逻辑**：
  1. 判断今天是否为周四，如果是则跳过
  2. 判断今天是否为交易日（排除节假日）
  3. 执行选股流程
  4. 生成Top 10推荐
  5. 推送给用户

### 排除周四的实现
```python
import datetime
today = datetime.datetime.now().weekday()  # 0=周一, 1=周二, 2=周三, 3=周四, 4=周五
if today == 3:  # 周四
    print("今日周四，跳过选股，避免周末持仓风险")
    exit(0)
```

## 五、胜率优化总结

1. **排除周末风险**：周四不选股，避免周五买入后持周末
2. **强化大盘过滤**：三指数+市场宽度多重验证
3. **板块效应**：只做热点板块，提高胜率
4. **综合评分**：多维度量化评估，避免单一指标误判
5. **严格排除**：新增新股、大宗折价等风险过滤
6. **动态止损**：次日开盘跌破买入价2%立即止损（需在卖出策略中体现）
