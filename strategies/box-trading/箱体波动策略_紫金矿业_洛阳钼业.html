⑤ 风控体系 · 四维校验

量能确认 + 大盘配合 + 仓位纪律 + 止损铁律

📊 成交量确认

箱底买入：缩量止跌（抛压衰竭）或放量长下影（资金抄底）
箱顶卖出：放量滞涨（主力出货）或冲高回落（假突破）
⚠️ 假突破过滤：突破箱体但无量 → 大概率假突破，不追涨不杀跌

🌍 大盘配合

大盘跌 + 个股跌到箱底 → 买点质量最高（系统性恐慌=黄金坑）
大盘涨 + 个股涨到箱顶 → 卖点质量最高（跟风冲顶=止盈时机）
大盘涨 + 个股不跟 → 警惕弱势，不宜加仓

⛔ 止损铁律

紫金：收盘跌破 27.50 → 全清
洛钼：收盘跌破 15.80 → 全清
不犹豫、不补仓、不幻想
连续止损3次 → 暂停策略，重新评估市场环境

📋 策略失效判定

1. 收盘价有效跌破止损线
2. 放量突破箱体上沿（紫金34.3 / 洛钼20.0）→ 趋势改变
3. 公司基本面重大变化（业绩暴雷/行业政策巨变）
4. 连续止损3次 → 策略不适配当前市场

🔄 箱体重新划定规则

当股价在止损线以上、但持续3天以上在箱体外运行时：

调整箱体上下沿
